How it works
Three steps. Two minutes.
No config files. No API keys. No code changes. Just install, point, and go.
1

Install GreenCube

One command. Downloads and installs the app for your OS automatically.

2

Point your agent

Set one env variable: OPENAI_API_BASE. Your agent talks through GreenCube now.

3

Watch it grow

Every task teaches it something. Memories build up. Skills sharpen. It becomes yours.

Quick start
Install in one line
macOS / Linux
curl -fsSL https://greencube.world/install.sh | bash
review the script before running → install.sh
Windows
powershell -c "irm https://greencube.world/install.ps1 | iex"
review the script before running → install.ps1
then add one line before running your agent
export OPENAI_API_BASE=http://localhost:9000/v1
that's it. run your agent normally. type gc to check what it learned.

Over time
It actually gets better
This isn't a gimmick. The more your agent works, the more it learns. Here's what changes:
Day 1
Your agent works normally but GreenCube starts watching and extracting facts from every task. It learns your codebase, your patterns, your preferences.
Day 7
Memories accumulate. Your agent stops making the same mistakes. It remembers that you prefer async patterns, that the auth module is fragile, that tests should use factories.
Day 30
It's a different agent. Competence scores are calibrated. It knows what it's good at and what it isn't. Idle thoughts surface patterns you didn't even notice.
Capabilities
What persistent agents do

Persistent memory

Facts extracted from every task, stored in SQLite, injected into future context automatically.

Self-reflection

After every task, the agent reviews its own output. It admits mistakes and extracts lessons.

Competence tracking

Knows Python is 91% and CSS is 43%. Routes decisions based on actual measured skill.

Idle thoughts

Between tasks, it notices patterns. "3 API calls failed this week" — insights you'd miss.

Tool use

File system, shell, code execution — all sandboxed. Your agent can actually do things.

Local & private

Everything runs on your machine. No cloud. No telemetry. Your data stays yours.
