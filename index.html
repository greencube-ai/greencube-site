maccurl -fsSL https://greencube.world/install.sh | bash
winpowershell -c "irm https://greencube.world/install.ps1 | iex"
thenexport OPENAI_API_BASE=http://localhost:9000/v1
Run your agent normally. Type gc to see what it learned.
What it does
What GreenCube does

Remembers everything

Facts from past tasks persist across sessions and get injected into future context automatically.

Self-checks its work

Scores its own output 1-5 after every task. Admits mistakes. Extracts lessons.

Tracks competence

Knows Python is 87% and CSS is 43%. More careful in weak areas, more confident in strong ones.

Prevents repeat mistakes

If you thumbs-down a response, it catches the same pattern next time and fixes it before you see it.

Thinks when idle

Between tasks, it notices patterns. "3 API calls failed this week." Insights you'd miss.

Runs locally

Everything on your machine. No cloud. No telemetry. SQLite database you own.
