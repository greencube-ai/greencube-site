set up in 60 seconds

seriously. three steps.

download and open GreenCube

the proxy starts automatically. lives in your system tray.

add one line before running your agent
export OPENAI_API_BASE=http://localhost:9000/v1

works with OpenClaw, LangChain, CrewAI, Python, curl — anything OpenAI-compatible

that's it. run your agent.

GreenCube handles the rest. check back anytime to see what your agent learned.

what your agent gets

  • memoryremembers across sessions. relevant context injected into every request automatically.
  • docker sandboxtool calls run in isolated containers. your machine stays clean.
  • audit trailevery action logged with timestamps, costs, and results.
  • streamingSSE passthrough. responses stream in real time like normal.
  • multi-providereach agent can use a different LLM. openai, ollama, whatever you want.
  • 8 built-in toolsshell, file read/write, http, reminders, agent messaging, and more.
experimental

alive mode

turn this on and your agents start learning from their own experience. they reflect after tasks, extract knowledge, and get smarter over time. uses extra tokens. off by default.

reflection — reviews what went well and what didn't
knowledge — extracts structured facts it can recall later
agent-to-agent — agents can ask each other for help
spawn specialist — creates a child agent when struggling
